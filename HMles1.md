#Подсказки по Git

Создание репозитория: 
```sh
git int 
```
Добавление файла к коммиту:
```sh 
git add
```
Создание коммита:
```sh
git comit -m
```
Вывод на экран истории всех коммитов с их хеш-кодами:
```sh 
git log
```
Переход от одного коммита к другому:
```sh
git checkout
```
Вернуться к актуальному состоянию и продолжить работу:
```sh
git checkout master
```
